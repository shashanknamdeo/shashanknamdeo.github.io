# Shashank Namdeo - Backend Portfolio

A premium, engineering-centric portfolio built with **Vite**, **Tailwind CSS**, and **Vanilla JS**.

## 🚀 Features
- **Architecture First**: Detailed project cards with API samples and system diagrams.
- **Terminal UI**: High-impact hero section for backend developers.
- **Data-Driven**: Projects are managed via `public/data/projects.json`.
- **Optimized**: SEO metadata, semantic HTML, and high performance.

## 🛠️ Usage

### Local Development
```bash
npm install
npm run dev
```

### Build for Production
```bash
npm run build
```

## 📁 Structure
- `src/`: Core logic and styles.
- `public/data/`: Project database (JSON).
- `public/assets/`: Static icons and PDFs (Resume).
